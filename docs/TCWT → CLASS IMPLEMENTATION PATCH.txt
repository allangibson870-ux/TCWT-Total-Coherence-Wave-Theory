3. GRAVITY MODULE REPLACEMENT (tcwt_gravity.c)
Replace Poisson solver:
ORIGINAL CLASS:
k2 * Psi = -4πG a^2 rho delta
TCWT VERSION:
double mu_k_a;

mu_k_a =
(1.0 + sqrt(k2 / (k2 + tcwt_kLambda*tcwt_kLambda)))
/
(1.0 + pow(k2,2) / pow(tcwt_Lambda,4) * pow(a,-4));
Modified Poisson equation:
Psi = -4.0 * M_PI * tcwt_Geff * a*a *
      (rho_m * delta_m + delta_rho_Omega + delta_rho_G)
      * mu_k_a;
Ensure:
Phi = Psi;

(no anisotropic stress introduced)

4. PERTURBATIONS MODULE (tcwt_perturbations.c)
