# TCWT – Hubble Tension Simulation Run (DESI-Aligned)

This run compares the TCWT resonant H(z) model with DESI-style reconstructed H(z) points.

## What this run shows

- DESI-style H(z) points with error bars.
- TCWT resonant H(z) curve with small, damped oscillations.
- SH0ES and Planck reference H₀ values.
- A direct visual comparison of TCWT vs observationally shaped data.

## Status

- This is a phenomenological TCWT H(z) ansatz.
- Parameters illustrate the mechanism rather than fit DESI/Planck.
- Next step: derive the functional form from the TCWT effective Friedmann equation.
