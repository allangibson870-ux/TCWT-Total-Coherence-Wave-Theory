# TCWT Simulation Run: 2026-03-06_1050 Entanglement Untangling Distance

**Run Description**  
Shows how entanglement fidelity decays with physical separation distance due to accumulated phase mismatch from the Ω gradient.

**Scientific Context**  
In TCWT, entangled particles stay coherent while their phases remain synchronised with the Hum.  
Separation introduces tiny Ω gradients → phase mismatch → fidelity drops.  
κ = 1.454 is derived from hydrogen squeeze leakage.

**Interpretation of the PNG**  
- Fidelity starts at 1.0 (perfect entanglement).  
- Drops smoothly with distance.  
- Crosses classical limit (0.5) at the marked **untangling distance**.

**Parameters Used**  
- κ = 1.454 (from hydrogen)  
- Coherence scale tuned to realistic drop-off
