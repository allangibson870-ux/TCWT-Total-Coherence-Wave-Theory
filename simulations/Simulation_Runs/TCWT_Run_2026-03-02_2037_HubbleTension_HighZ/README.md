# TCWT – Hubble Tension Simulation Run (High-z Extension)

This run extends the TCWT phenomenological H(z) model out to z = 20.

- This is a phenomenological TCWT H(z) ansatz, not a DESI/Planck best-fit.
- Parameters illustrate the mechanism rather than minimize χ².
- Next step: derive this form from the TCWT effective Friedmann equation.

The curve:
- Starts near the SH0ES value (~73 km/s/Mpc)
- Shows a small, gently damped oscillation (~1 km/s/Mpc)
- Slowly decays toward the Planck value (~67.4 km/s/Mpc)
- Asymptotes correctly at high z without overshooting
